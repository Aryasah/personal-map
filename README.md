# Mappa
##### A map app made with amazing features.


### Technologies Used
* HTML5
* CSS3
* Javascript
* MapboxGl


*Key Features*
* Supports three types of maps.
  * Satellite Map.
  * Street Map.
  * Traffic Map.
* Suppoorts Both Dark and Light Mode.
* Responsive on mobile.
